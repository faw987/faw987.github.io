User Guide

What is this?

GPX Viewer is a single-page web app for visualizing one or more GPX track files on an interactive map. Everything runs locally in your browser — your files are never uploaded anywhere.

Loading GPX files

  • Drag & drop one or more .gpx files anywhere onto the map.
  • Or click Open GPX files in the header to pick files from a dialog.
  • You can load multiple files; each track is drawn in a different color and listed in the legend (bottom-left).
  • The map automatically zooms to fit all loaded tracks.

Supported GPX content

  • <trk> with <trkseg> / <trkpt> (standard recorded tracks).
  • <rte> with <rtept> (planned routes).

Changing the map style

Use the layer control in the top-right corner of the map to switch base layers:

  • Streets (OSM) — standard OpenStreetMap.
  • Terrain (OpenTopoMap) — contour lines and relief shading, good for hiking and cycling.
  • Humanitarian (OSM HOT) — a high-contrast alternative emphasizing footpaths and facilities.
  • Satellite (Esri) — aerial imagery.

Map navigation

  • Pan: click and drag.
  • Zoom: mouse wheel, pinch, or the + / - buttons.
  • Click a track to see its filename in a popup.

Menu actions

  • User Guide — opens this help.
  • Clear All Tracks — removes every loaded track from the map.

Privacy

GPX parsing happens entirely in your browser. Map tiles are fetched from the selected tile provider (OpenStreetMap, OpenTopoMap, or Esri). No track data is transmitted.

Tips

  • Drop a folder-worth of GPX files at once to compare multiple rides or hikes side-by-side.
  • Switch to the Terrain layer to see elevation context for mountain routes.
